## 📝 使用方法

### 单个视频推理

```bash
python scripts/inference/infer_video.py \
  --model runs/train/v11_material_9class_20260306_115502/weights/best.pt \
  --video /path/to/video.mp4 \
  --output runs/inference/v11_videos \
  --conf 0.3 \
  --iou 0.45
```

### 批量视频推理

```bash
python scripts/inference/infer_video.py \
  --model runs/train/v11_material_9class_20260306_115502/weights/best.pt \
  --video /path/to/video/directory/ \
  --output runs/inference/v11_videos \
  --conf 0.3 \
  --iou 0.45
```

### 调整置信度阈值

```bash
# 更严格（更少误报）
--conf 0.5

# 更宽松（更多检测）
--conf 0.2
```
